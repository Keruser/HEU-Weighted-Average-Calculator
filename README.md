# 成绩均分计算器 - 多学期对比版

一个简单易用的成绩均分计算软件，支持Excel文件导入，自动计算学分加权平均分，并提供**多学期成绩对比分析**功能。

## ✨ 功能特点

- 📊 **自动计算**：上传Excel成绩表，一键计算均分
- 🎯 **智能识别**：自动识别公选课、专选课并排除
- 📝 **等级转换**：考查课自动转换（优秀=95，良好=85，中等=75，及格=65，不及格=0）
- 📈 **多学期对比**：自动识别「学年学期」列，支持多学期成绩趋势分析
- 🧮 **总均分计算**：一键计算所有学期的总加权平均分、总学分、总课程数
- 📉 **趋势分析**：计算学期间成绩变化、稳定性分析、最高分/最低分追踪
- 📋 **详细报告**：生成包含总均分评价的详细分析报告
- 💾 **结果保存**：支持保存计算结果为JSON格式
- 🖥️ **桌面应用**：独立的exe程序，无需安装Python

## 🚀 使用方法

### 1. 解压后直接使用（已打包）

双击运行 `成绩均分计算器.exe` 即可。

## 📋 Excel文件格式要求

您的成绩Excel文件需要包含以下列：


| 列名         | 说明                       | 示例               |
| ------------ | -------------------------- | ------------------ |
| **学年学期** | 学期标识（用于多学期识别） | 2024-2025学年1学期 |
| 课程名       | 课程名称                   | 高等数学           |
| 总成绩       | 原始成绩                   | 85 或 "优秀"       |
| 学分         | 课程学分                   | 3.0                |
| 课程类别     | 用于识别公选课             | 24通识教育必修课   |
| 课程性质     | 必修/选修                  | 必修               |
| 校公选课类别 | 公选课标记                 | 为空表示非公选课   |
| 考试类型     | 考试/考查                  | 考试               |
| 是否有效     | 是否计入                   | 是/否              |

> 💡 **提示**：程序会自动根据「学年学期」列识别不同学期，无需手动分离文件。
>
> 一般就是从教务官方导出的excel文件，无需修改

## 📊 界面说明

### 单学期计算页

1. **选择文件**：点击"选择文件"按钮，选择成绩Excel文件
2. **学期选择**：如果文件包含多个学期，可在下拉框中选择：
   - **特定学期**：计算该学期的均分
   - **全部学期**：计算所有学期的总加权平均分（推荐）
3. **计算均分**：点击"计算均分"按钮
4. **查看结果**：
   - 顶部显示平均分、课程数、学分统计
   - 显示最高分和最低分课程
   - 表格显示每门课程的详细信息和是否计入（选择"全部学期"时会显示所有学期课程）
5. **保存结果**：导出当前计算结果（单学期或全部学期）

### 多学期对比页

1. **刷新对比数据**：计算完成后，点击此按钮生成分期对比
2. **查看概览**：表格展示所有学期的关键指标对比，**最后一行显示"全部学期"的总统计**
3. **详细分析**：文本区域显示：
   - 📊 **总均分概览**：所有学期的总加权平均分、总学分、总课程数
   - 📈 **平均分趋势**：各学期平均分及相比上学期变化
   - 📚 **学分修读情况**：累计学分及各学期分布
   - 🏆 **高低分课程**：各学期的最高分和最低分课程
   - 📝 **总结建议**：表现最佳学期、需要关注的学期、总均分评价、成绩稳定性分析
4. **导出报告**：将对比分析保存为文本文件

## 🧮 计算规则

### 1. 均分计算方式

- 使用**学分加权平均法**
- 公式：`Σ(成绩 × 学分) / Σ学分`

### 2. 排除的课程

以下课程**不计入**均分计算：

- ❌ 公选课（校公选课类别不为空）
- ❌ 专选课（课程类别或课程性质包含"选修"）
- ❌ 无效课程（是否有效 ≠ "是"）

### 3. 考查课等级转换

如果课程为**考查课**，成绩按以下规则转换：


| 等级   | 转换分数 |
| ------ | -------- |
| 优秀   | 95       |
| 良好   | 85       |
| 中等   | 75       |
| 及格   | 65       |
| 不及格 | 0        |

> 注：考试课按原始分数计算，不进行转换。

## 💡 使用示例

### 单学期计算示例

假设您有以下成绩：


| 课程名   | 成绩 | 学分 | 课程性质 | 考试类型 | 是否计入    |
| -------- | ---- | ---- | -------- | -------- | ----------- |
| 高等数学 | 85   | 4    | 必修     | 考试     | ✓          |
| 大学英语 | 优秀 | 3    | 必修     | 考查     | ✓ (转为95) |
| 音乐欣赏 | 90   | 2    | 选修     | 考查     | ✗ (公选课) |

计算结果：

- 计入课程：高等数学、大学英语
- 加权均分：(85×4 + 95×3) / (4+3) = 89.29

### 多学期对比示例

上传包含多个学期的成绩文件后：

```
【平均分趋势】
  2024-2025学年1学期: 90.38 分 （基准学期）
  2024-2025学年2学期: 93.13 分 （↑ 较上学期提升 2.75 分）
  2025-2026学年1学期: 93.93 分 （↑ 较上学期提升 0.80 分）

整体趋势：从首学期到末学期，平均分上升了 3.55 分
```

## ❓ 常见问题

**Q: 软件无法打开？**
A: 请确保您的系统是Windows 10或更高版本，并安装了最新的Visual C++运行库。

**Q: Excel文件读取失败？**
A: 请检查Excel文件是否符合格式要求，确保包含"课程名"、"总成绩"、"学分"、"学年学期"等必要列。

**Q: 某些课程没有被排除？**
A: 请检查"课程类别"和"校公选课类别"列的值是否正确。

**Q: 考查课没有转换？**
A: 请确保"考试类型"列的值为"考查"，且成绩为"优秀"、"良好"等文字等级。

**Q: 多学期对比显示不完整？**
A: 请确保Excel文件中的「学年学期」列格式统一，例如："2024-2025学年1学期"。

**Q: 如何查看特定学期的详情？**
A: 在"单学期计算"页选择学期下拉框，选择目标学期后点击"计算均分"。
